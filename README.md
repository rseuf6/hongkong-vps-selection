说实话，香港VPS市场水很深。

打着"香港"旗号的产品，实际上线路差异极大。有的是真正的香港本地机房，有的是绕了一大圈的中转，延迟标注得好看，实际ping值一测就露馅。更麻烦的是，很多商家的"香港"节点在晚高峰时段会严重拥堵，白天测试完美，晚上8点以后卡成PPT。
